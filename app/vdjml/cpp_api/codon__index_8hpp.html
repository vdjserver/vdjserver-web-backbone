generate enum for all amino acid codons from a nucleotide enum

Each enum value is formed by combining three nucleotide enum values. Nucleotide enum may include wildcards and other values. All combinations of nucleotide enum values should be supported.

Parameters
nameenum name
nucssequence of nucleotide letters and values
maxmaximal value of nucleotide enum; used to define largest possible codon value, to ensure that codon enum supports all possible values
